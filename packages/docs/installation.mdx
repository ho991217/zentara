## Install the component

Install the Zentara core package in your project with any package manager you prefer.

<CodeGroup>

```bash pnpm
pnpm add @zentara/core
```

```bash npm
npm install @zentara/core
```

```bash yarn
yarn add @zentara/core
```

</CodeGroup>

<Info>
  **Requirement**: React (version 18 or higher) and ReactDOM (version 18 or
  higher) are required to use the component.
</Info>
