## Properties

`@zentara/plugin-suggestions` provides the following properties:

<ResponseField name="triggers" type="string[]" required>
The triggers to show suggestions.

Example: `['{', '{{', '{{.']`

</ResponseField>

<ResponseField name="suggestions" type="string[]" required>
The suggestions to show.

Example: `['foo', 'bar', 'baz']`

</ResponseField>

<ResponseField name="transform" type="(selelcted: string) => string">
The transform function to transform the suggestion.

When the user selects a suggestion, the transform function is called with the selected suggestion and the transform function returns the transformed suggestion.

```tsx example
// when the user selects 'foo',
// suggestionPlugin inserts '{{.foo}}' into the input
(selelcted) => `{{.${selelcted}}}`;
```

</ResponseField>

<ResponseField name="renderSuggestion" type="(suggestion: string) => ReactNode">
The render function to render the suggestion. User can decide how to show the suggestion list.

```tsx example
(emojiName) => <div>{emojis[emojiName]}</div>;
```

</ResponseField>

<ResponseField name="maxSuggestions" type="number" default={5}>
The maximum number of suggestions to show.

Example: `10`

</ResponseField>

<ResponseField name="suggestionsListOffset" type="number" default={0}>
The offset of the suggestions list.

Example: `8`

</ResponseField>
